# Advanced Configuration & Patches

This section contains advanced configuration guides and community-developed patches for the Agent Client plugin.

::: warning
Most of these guides involve patching the `main.js` file of the plugin. These changes will be overwritten when the plugin is updated and must be re-applied.
:::

## Available Guides

- [Agent Settings Reference](agent-settings): A reference for `data.json` configuration for various AI agents.
- [Custom Agent Logos](custom-logo): How to add custom icons for different agents in the chat header.
- [ESC Shortcut Patch](esc-shortcut): Adding a keyboard shortcut to stop the agent's response.
- [Windows Installation](windows-setup): A guide for installing and configuring Agent Client on Windows.
- [Usage Display Patch](usage-display-patch): Fixing issues with the context usage display.
- [Rich Text Copy Fix](rich-text-copy-fix): Restoring the ability to copy rich text from the chat.
- [Responding Status Fix](responding-fix): Fixing a bug where the "Responding..." status persists after the chat ends.
- [RAG Setup Guide](rag-setup): Instructions for reconfiguring the RAG (Retrieval-Augmented Generation) features.
